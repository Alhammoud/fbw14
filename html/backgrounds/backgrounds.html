CSS Hintergründe

Eigenschaften

Es gibt einige verschiedene Hintergrund Eigenschaften:

background-image
Diese Eigenschaft ist die Location von dem Bild, zum Beispiel h1 { background-image: url('http://website.com/img/background.jpg'); } oder relativer Pfad ohne https: h1 { background-image: url('img/background.jpg'); }
background-color
Diese Eigenschaft ist die Farbe vom Hintergrund, zum Beispiel h1 { background-color: pink; } oder alternativ als h1 { background-color: #FFC0CB; }
